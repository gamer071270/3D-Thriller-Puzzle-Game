# 3D Thriller Puzzle Game - Unity Project (Code Only)

This repository contains the core gameplay codes of a narrative-driven thriller puzzle game developed in Unity (C#).
The game challenges players to solve puzzles revealing the character's past to reach the end.

The project focuses on:
- Environmental storytelling  
- Logic-based puzzles  
- Player interaction systems  

Note: This repository includes only scripts.  All assets (3D models, textures, audio, animations, UI visuals) are intentionally excluded.

---

## Game Overview

The game is a first-person thriller puzzle experience where the player explores a house and solves story-driven puzzles.

Implemented systems include:
- Interactable objects (doors, drawers, wardrobes, items)
- Raycast-based interaction system
- Puzzle logic (multi-step, state-based puzzles)
- Inventory & hand-held object mechanics
- Event-driven architecture for puzzle progression

---

## Implemented Puzzles

- Photo Puzzle : Collecting and combining photo fragments to reveal a picture, then placing specific objects in correct locations using the picture.

- Fuse Box Puzzle : Managing electrical load by activating devices without overloading the system.
